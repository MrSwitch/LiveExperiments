Scopes

Scope Description

wl.applications

Enable read access to a user's applications registered with Messenger Connect.

wl.applications_create

Enables the registration of new applications with Messenger Connect on behalf of a user.

wl.basic

Enables read access to a user's public and basic profile information such as gender.

Enable read access to a user's list of contacts and friends.

wl.birthday

Enables read access to a user's birthday information including birth day, month, and year.

wl.contacts_birthday

Enables read access to a user's contacts' birth day and birth month.

wl.contacts_photos

Enables read access to a user's friends' photos, videos, and albums.

wl.emails

Enables read access to a user's personal, preferred, and business email addresses.

wl.events_create

Enables the creation of events on a user's calendar.

wl.offline_access

Enables access to a user's information, and creating new objects on behalf of a user, after he or she has ended the corresponding web browsing session.

wl.phone_numbers

Enables access to a user's personal, business, and mobile phone numbers.

wl.photos

Enables read access to a user's photos, videos, and albums.

wl.postal_addresses

Enables read access to a user's postal addresses.

wl.share

Enables the creation of activities on a user's activity feed and personal status message.

wl.signin

Enables automatic site sign-in for users who are already signed in to Messenger Connect.

wl.work_profile

Enables read access to a user's employer and work position information.
