Live Experiments | Start

This no bullshit guide takes you from zero to a Windows Live authentication system on your website.

Step 1: Register your domain/site

  1. Go to http://manage.dev.live.com and signin
  2. Find and click on the link "Create Application" and provide information about your environment domain
  3. Save the form and make a note of the "Client ID"

Step 2: Build a simple test script

Pop your client_id in the script below, save the script so you can run it from the domain you registered.

<!DOCTYPE html>
<html>
<head>
	<script src="http://js.live.net/v5.0/wl.js"></script>
</head>
<body>
	<div id="signin"></div>
<script>
	WL.init({
		client_id: CLIENT_ID,
		redirect_uri: window.location.href
	});
	WL.ui({ name: "signin", element: "signin", scope : "wl.basic" });
</script>
</body>
</html>

Now fire it up on your dev enviroment. The button on this page will trigger the consent flow and the text will change from "Connect" to "Sign out".... so you know your signed in.

Step 3: Now go!

Explorer the demo's under the 5.0 column on the homepage and uses the forums links at the bottom if you can't find something.
